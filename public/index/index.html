ANNOT-GENES

WEB visualization tool for functional annotation of genes


Goal of this site

The broadband sequencing contributes largely to the biological study by obtaining a significant number of information on the genome. The display of the note on genes is important for the exploitation of the data, she allows to see in clear and structured way the set of notes attributed to a cluster of genes. It is in this purpose that this Web interface was developed. It has for objective to allow generate a treemap from data in the format CSV which are converted in format Json. The data in format CSV must be structured in a way to allow their conversion in a efficient way.
