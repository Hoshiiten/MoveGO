ANNOT-GENES

WEB visualization tool for functional annotation of genes


How to use ANNOT-GENES

ANNOT-GENES will help you to visualize results of gene annotation via an interactive treemap. Our website simply takes as input a list of annotation clusters and a list of genes associated to each terms. Then returns a dynamic treemap of it. Here, we will explain fonctionnalities offered by our website and how to use them.


Input

In a first time, you have to get a csv format file. This file should contain clusters of terms in a specific format. Here is an example of file format needed :



You have to upload this csv file on the website to obtain a treemap of your clusters.


Dynamic Treemap

The dynamic treemap obtained represents all clusters. A cluster is represented by a big colored rectangle. Each cluster has a different color. In each cluster you could find all the terms presents in the cluster.

  • If you click on a specific cluster, information about the cluster will display below treemap. Hence, you could know all the terms and all the genes annotated for this specific cluster.
  • If you click on a specific term, different information will display. Hence, you could discover the Gene Ontology ID of this term, theirs ICnuno, ICzhou, parents, children and genes annotated for this term.

Export

You have the possibility to export the resulting treemap. This export will allow you to see the treemap locally without having need to get connected to the internet. A README would be provided in the downloaded folder to help you obtaining your treemap.
